## Development

Install [pnpm](https://pnpm.io/installation) and [Hatch](https://hatch.pypa.io/latest/install/) to set up the development environment.

```shell
git clone https://github.com/gnueaj/mu-dashboard.git && cd mu-dashboard/frontend
pnpm install
hatch shell
pnpm dev
```
